# Desktop Stopwatch Application Using Python Library Tkinter

## Description

A simple desktop stopwatch built with Python and Tkinter. The application provides an intuitive graphical interface for measuring elapsed time with Start, Stop, and Reset controls.

---

## Features

- Start stopwatch
- Stop stopwatch
- Reset stopwatch
- Live HH:MM:SS timer display
- Responsive GUI
- Uses Tkinter's event scheduling
